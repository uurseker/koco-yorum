# Koço Restaurant — Yorum Oluşturucu

Koço Restaurant müşterileri için Google yorum oluşturma aracı.

## Kurulum

### Seçenek 1: Wix'e Gömme (Önerilen)
1. Wix editöründe yeni bir sayfa oluşturun: `/yorum`
2. Sayfaya **Embed HTML** (HTML Yerleştir) widget'ı ekleyin
3. `index.html` dosyasının içeriğini yapıştırın
4. Yayınlayın → `kocorestaurant.com/yorum` adresinden erişilebilir

### Seçenek 2: GitHub Pages (Ücretsiz Alternatif)
1. Bu repoyu GitHub'a push edin
2. Settings → Pages → Source: main branch
3. Custom domain: `yorum.kocorestaurant.com` (DNS'te CNAME kaydı gerekli)

### Seçenek 3: Vercel / Netlify
1. Repoyu bağlayın
2. Custom domain ayarlayın
3. Otomatik deploy

## QR Kod
`qr-koco.png` dosyasını masalara, hesap zarflarına veya menüye basabilirsiniz.

## Baskı Kartı
`baski-karti.pdf` dosyasını A6 veya kredi kartı boyutunda bastırabilirsiniz.
